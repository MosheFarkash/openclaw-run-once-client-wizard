# Brand Consistency and Refresh Modes

## Brand Consistency Mode

Use when the user has a website, brand reference, previous creative batch, or saved brand brief.

Goal: make new creatives feel like they belong to the same brand while still testing new performance angles.

Preserve:

- Core color palette.
- Typography feel.
- Spacing and layout personality.
- Visual tone.
- Level of polish/premium feel.
- Text density norms.

Vary:

- Hook.
- Angle.
- Proof.
- CTA.
- Image composition.
- Offer framing.

Do not re-analyze the brand from scratch on every run if a recent brand brief exists. Reuse saved brand memory unless the user says the brand changed.

## Ad Fatigue / Refresh Mode

Use when a campaign manager or user says a creative used to work but performance is dropping.

Goal: refresh attention without losing the winning insight.

Preserve:

- Winning promise.
- Winning audience insight.
- Product/offer clarity.
- Any proven proof element.

Change:

- Visual hook.
- First line / headline.
- Framing metaphor.
- Layout pattern.
- Background or focal image.

Create two refresh tiers:

1. Close Refresh: same concept, new hook/layout variation.
2. Far Refresh: same underlying promise, visibly different creative pattern.

## Winner Variation Mode

Use when a specific creative is winning.

Generate 3-5 variations that change only one major variable at a time:

- Text hook variation.
- Proof variation.
- Offer framing variation.
- CTA variation.
- Visual emphasis variation.

Keep everything else as identical as possible.

## Losing Pattern Avoidance

When the user or campaign manager identifies losers, record the pattern.

Examples:

- Too generic.
- Too much text.
- Low-quality leads.
- Wrong audience framing.
- Cheap/discount-heavy tone.
- Too abstract.
- Weak product clarity.

Avoid repeating losing patterns in future batches.
